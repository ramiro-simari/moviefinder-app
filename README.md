# MovieFinder

MovieFinder is a responsive web application that allows users to search for movies, explore featured collections, and view detailed information about each title.

Built with vanilla JavaScript, HTML, and CSS, the project uses the OMDb API to fetch movie data in real time.

## Live Demo

https://moviefinder-app.vercel.app/

## Features

- Search movies by title
- Real-time search with debounce
- Featured movie collection on page load
- Quick category filters
- Movie details modal
- Pagination support
- Skeleton loading states
- Responsive design for desktop, tablet, and mobile devices
- Direct link to official trailers on YouTube
- Accessible keyboard navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- OMDb API
- Vercel

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/moviefinder.git
```

2. Navigate to the project folder

```bash
cd moviefinder
```

3. Open `index.html` in your browser
